### 数据帧结构

| **帧头**     | **原始数据长度** | **原始数据** | **CRC 校验**   |
| ------------ | ---------------- | ------------ | -------------- |
| 2 字节       | 2 字节           | N 字节       | 1 字节         |
| `0x55, 0xAA` | 高位、低位       | 原始数据内容 | CRC-8 校验字节 |

### 示例

| **帧头**     | **原始数据长度** | **原始数据**                        | **CRC 校验** |
| ------------ | ---------------- | ----------------------------------- | ------------ |
| `0x55, 0xAA` | `0x00, 0x03`     | `0x01, 0x02, 0x03`                  | `0x84`       |
| `0x55, 0xAA` | `0x00, 0x03`     | `0x7D, 0xAA, 0x55`                  | `0x28`       |
| `0x55, 0xAA` | `0x00, 0x06`     | `0x01, 0x02, 0x03 0x7D, 0xAA, 0x55` | `0xEE`       |
